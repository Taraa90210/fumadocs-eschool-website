This page is used to add, view, edit, and delete instructional languages for each grade level and major.

The **Instructional Language** page is used to manage the list of instructional languages used in teaching and learning activities at school. This feature helps schools in determining the main language used for each subject, program, or specific curriculum.

<img src="/images/bahasa-pengantar.avif" alt="Image description" />



### Adding an Instructional Language

To add a new language:

1. Navigate to the **"Add Instructional Language"** form on the left side of the page.
2. Enter the **Name** of the instructional language, for example: `ENG` for English or `ID` for Bahasa Indonesia.
3. Click the **Save** button.
4. The language will automatically be added to the list on the right.

---

### Instructional Language Registry

The right section contains a list of all languages that have been added. This table displays:

- **No**: Sequence number.
- **Name**: Language name.
- **Actions**: Menu button to edit or delete data.

---

### Managing Language Data

Click the vertical three-dot icon (â‹®) in the **Actions** column on each row to reveal two options:

#### **Edit**

You can change the language name, then click **Save** to update the data.

#### **Delete**
This option will directly remove the language data from the system after confirmation.

> [!CAUTION]
> Ensure you only delete data that is no longer in use, as this may affect curriculum settings or other connected academic data.

---

### Usage Tips

- Use standard abbreviations for consistency (e.g., `ID`, `ENG`, `AR`).
- You can search for language data using the **Search** column in the upper right of the table.
- Click the **All** or **Deleted** button above the table to display active languages or those that have been deleted (if restore/archive features are available).

---

> [!NOTE]
> The Instructional Language module is an essential component for supporting multilingual learning management in school environments with special programs such as international or bilingual classes.
