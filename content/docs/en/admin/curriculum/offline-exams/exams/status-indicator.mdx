# Status Indicator

The Status Indicator feature functions to display information about the implementation status of previously scheduled exams. Through this feature, users can monitor whether exams have been completed according to the set schedule.

---

### How to View Exam Status Indicator

**1. Access Offline Exams → Exams Menu**

The Exam List table will display the list of exams that have been created.

**2. Click the Three-Dot Icon (⋮) in the Action Column**

Select the exam whose status you want to check, then click the (⋮) icon in the Action column.

**3. Select the "Status Indicator" Option**

From the dropdown menu that appears, click the "Status Indicator" option.

---

### Status Indicator Modal Display

After selecting Status Indicator, the system will display a modal containing details of exam implementation status with the following information:

| Column | Description |
|--------|-------------|
| **No** | Sequential number of registered exam schedule. |
| **Subject** | Name of the subject being tested. |
| **Date** | Date of exam implementation. |
| **Start Time** | Time when the exam begins. |
| **End Time** | Time when the exam ends. |
| **Subject Teacher** | Name of the teacher in charge of the relevant subject. |
| **Status** | Shows whether the exam has been **Completed** or is still in progress. |

**Note**: This status is informative and cannot be changed manually through this modal. The system will automatically mark the exam as "Completed" if the schedule has passed the set implementation time.


---

### Status Indicator Functions:

- Provide real-time information to admins or operators regarding exam status.
- Facilitate checking which exams are completed and which are still active or not yet implemented.
- Serve as a reference to proceed to the next stage, namely exam result publication.

---

### Brief Status Indicator Flow:

1. **Create Exam** → **Schedule** → **Execute**.
2. Check exam status through the **Status Indicator** option.
3. If the status is **"Completed"**, you can proceed to the result publication process.
