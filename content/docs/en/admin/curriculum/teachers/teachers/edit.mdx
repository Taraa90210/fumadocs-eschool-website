The Edit Teacher feature allows admins to update information of teachers already registered in the system. Either due to data changes (for example phone number, address, qualification), or for previous input corrections.

Quick access, instant changes. This feature is designed so that the teacher data management process remains neat and efficient.

---

### How to Access Edit Teacher Feature

1. Go to **Teachers** > **Teacher List** menu.
2. Find the teacher you want to edit.
3. Click the three-dot icon (⋮) in the **Action** column.
4. Select **Edit**, then the Edit Teacher window will appear as shown in the image above.

---



### Edit Teacher Form Components

| Field | Description |
|-------|-------------|
| **Email** | Cannot be changed as it is used as account identity. |
| **Full Name** | Required. Official name according to documents. |
| **Nickname** | Optional. Used for informal addressing. |
| **Gender** | Choose between Male or Female. |
| **Phone Number** | Make sure the number is active and valid for communication. |
| **Date of Birth** | Format: dd-mm-yyyy. Required. |
| **Image** | Upload teacher photo with ideal size (308px × 338px). |
| **Qualification** | Example: S1, S2, PNS, PPPK, or others. |
| **Current Address** | Current residence domicile. |
| **Domicile Address** | Can be filled differently if residence and domicile addresses are not the same. |
| **Salary** | Monthly salary or honorarium according to policy. |
| **Reset Password** | Check this box to reset the password for that teacher's account. |

> [!IMPORTANT]
> All required fields must be completed before clicking **Save**.

---

### Important Notes

- **Email cannot be changed** from this page. If there is an error, admin must delete and re-add the teacher.
- The **Reset Password** button will automatically reset the password and send instructions to email if the system is connected to an email delivery service.
- Data changes will be immediately reflected in the system, so make sure the data entered is correct before pressing **Save**.

---

### Usage Tips

- Use this edit feature regularly to ensure teacher data remains up-to-date.
- Specifically for status changes such as active/inactive, do so through the main Add/Teacher List form.
- If the photo does not appear after upload, make sure the file format is `.jpg` or `.png` and does not exceed the size limit.
