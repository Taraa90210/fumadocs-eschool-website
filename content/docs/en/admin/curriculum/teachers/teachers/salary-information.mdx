The Salary Information feature allows admins to set and manage teacher income details, including basic salary, allowances, deductions, to net salary calculations. This information is sensitive and can only be accessed by authorized parties such as school treasurers or HR admins.

---

### How to Access

1. Open **Teachers** > **Teacher List** menu.
2. Find the teacher whose salary information you want to view or set.
3. Click the three-dot icon (â‹®) in the **Action** column.
4. Select the **Salary Information** option.
5. The system will display a form as shown in the image above.

---

### Salary Form Explanation

#### **Allowances Section**

- **Allowance Type**: Select from the list (example: Holiday, Certification, Position).
- **Amount (Amt)**: Nominal amount of allowance.
- Use the **+ New Allowance** button to add additional entries.

#### **Deductions Section**

- **Deduction Type**: For example Tax, Attendance Deduction, or period-based deduction.
- **Percentage**: Percentage deduction from basic salary or total.
- Use the **+ New Deduction** button to add other entries.

---

### Automatic Calculation Results

After all data is entered, the system will automatically calculate:

| Component | Description |
|-----------|-------------|
| **Basic Salary** | Main salary received by teacher. |
| **Allowances** | Total of all salary additions. |
| **Deductions** | Total deductions based on percentage or fixed amount. |
| **Net Salary** | Final salary received by teacher after deductions and added allowances. |

**Example:**
- Basic Salary: 55
- Allowances: 20,000
- Deductions: 10% of 55 = 5.5
- **Net Salary** = 55 + 20,000 - 5.5 = **20,049.5**

---

### Usage Tips

- Valid salary data will facilitate the recording and disbursement of honorariums.
- Make sure all numbers are filled in correctly, and allowances/deductions do not overlap.
- Click **Save** after finishing inputting or updating data.

---

### Security Notes

> [!CAUTION]
> - This feature can only be accessed by users with certain privileges (Admin / Treasurer / Principal).
> - This information cannot be accessed by the teacher concerned unless given special access.
