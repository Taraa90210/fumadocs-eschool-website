### Menu Access

Access the **Sliders** menu from the admin sidebar. On this page, you can see two sections:
- Add slider form at the top.
- Slider list table at the bottom.

---

### Adding a New Slider

1. Click on the **Add Sliders** form at the top of the page.
2. Fill in the columns:
   - **Image**: Required to upload. Click the **Upload** button to select an image from your device.
   - **Link**: Optional, but recommended if the slider should direct to a specific page.
3. Click the **Save** button to save the slider.

> [!IMPORTANT]
> Image upload is mandatory, otherwise the system will reject the save.

---

### Editing a Slider

<img src="/images/edit-sliders.png" alt="Image description"  />

1. Find the slider data you want to edit in **Slider List**.
2. Click the three-dot vertical icon (⋮) in the **Action** column.
3. Select the **Edit** option.
4. The system will display a form with pre-filled data.
5. Make desired changes (replace image, change link, etc.).
6. Click **Save** to save changes.

> [!TIP]
> Sliders can be updated if there are information revisions or new images.

---

### Deleting a Slider

1. Find the slider you want to delete in the list.
2. Click the **Action** button (⋮) on that row.
3. Select the **Permanent Delete** option.
4. The system will request confirmation before deletion is performed.

> [!CAUTION]
> Deletion is permanent and cannot be undone.

---

### Slider List Display

<img src="/images/daftar-sliders.png" alt="Image description"  />

- Refresh icon ↻ to update the list.
- Layout/grid icon to change display.
- Download icon ⬇️ to export slider data.

---

### Usage Tips

- Use horizontal (landscape) sized images to fit the slider layout.
- Avoid using large images to prevent slow loading.
- Ensure added links are active and valid.
