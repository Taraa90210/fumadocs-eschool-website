Page for managing email templates sent automatically by the system.

---

### Main Function

Set and adjust email templates for various automatic system notifications.

---

### Email Template Types

#### Student/Parent Notifications
- **Registration Success**: Welcome email for new students
- **Reset Password**: Email for password reset
- **New Billing**: Payment billing notification
- **Payment Success**: Payment confirmation
- **Announcement**: Template for school announcements

#### Teacher/Staff Notifications
- **New Account**: Email for new teacher/staff accounts
- **New Task**: Notification of assigned tasks
- **Leave Approved/Rejected**: Leave application status
- **Salary**: Monthly salary slip

#### Admin Notifications
- **Daily Report**: Summary of daily activities
- **System Alert**: Important system warnings

---

### Editing Templates

1. Select the template you want to edit
2. Edit components:
   - **Subject**: Email title
   - **Body**: Email content (HTML supported)
   - **Variables**: Use dynamic variables like `{student_name}`, `{date}`, etc.
3. **Preview**: View email layout
4. **Test**: Send a test email
5. **Save**: Save changes

---

### Available Variables

Dynamic variables that can be used:
- `{name}` - Recipient name
- `{email}` - Recipient email
- `{school}` - School name
- `{date}` - Current date
- `{amount}` - Nominal (for billing/payment)
- And other variables according to context

---

### Usage Tips

- Use polite and professional language
- Test templates before activating
- Ensure variables are written correctly
- Include school contact information in the footer
