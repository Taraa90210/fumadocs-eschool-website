Page for configuring payment methods and payment gateway integrations.

---

### Main Function

Manage various payment methods available for parents/students to perform school fee payments.

---

### Payment Methods

#### Manual Payment
- **Cash**: Direct payment at school
- **Bank Transfer**: Transfer to school account
- **Check/Giro**: Payment using checks

#### Payment Gateway
- **Midtrans**: Integration with Midtrans
- **Xendit**: Integration with Xendit
- **DOKU**: Integration with DOKU
- **Others**: Other payment gateways

---

### Gateway Configuration

For each payment gateway, set:
- **API Key**: API key from the provider
- **Merchant ID**: School merchant ID
- **Environment**: Production or Sandbox
- **Status**: Active or Inactive

#### Additional Settings
- Admin fees (if any)
- Supported payment methods (credit card, e-wallet, etc.)
- Automatic payment notifications

---

### School Bank Accounts

Add account information for manual transfer:
- Bank Name
- Account Number
- Account Holder Name
- Branch

---

### Usage Tips

- Activate at least one payment method
- Test configuration in sandbox environment first
- Inform parents about available payment methods
- Monitor transactions regularly
