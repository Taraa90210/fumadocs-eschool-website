# Refunds

Page for managing payment refund policies and processes.

---

### Main Function

This page is used to:
- Set refund policies
- Manage refund requests from parents/students
- Process approved refunds
- Record refund history

---

### Policy Settings

#### Refund Requirements
- Refund application deadline
- Required documents
- Acceptable reasons
- Refund percentage

#### Approval Process
- Required approval levels
- Maximum processing time
- Refund method (transfer, cash)

---

### Managing Refund Requests

1. **View Request List**
   - Status: Pending, Approved, Rejected
   - Filter by date, student, or amount

2. **Process Requests**
   - Review supporting documents
   - Verify eligibility
   - Approve or reject with reasons

3. **Recording**
   - Record refund details
   - Update student payment status
   - Generate proof of refund

---

### Usage Tips

- Establish clear and consistent policies
- Document all refunds properly
- Communicate the process to parents
