Page for managing school profile and identity information.

---

### Purpose

Provides an interface to set and update basic school information displayed throughout the system.

---

### Components and Settings

#### Basic Information
- **School Name**: Official institution name
- **Address**: Full school address
- **Phone Number**: School contact
- **Email**: Official school email
- **Website**: School website URL

#### Visual Identity
- **School Logo**: Upload official logo (format: jpg, png, svg)
- **Favicon**: Browser tab icon
- **Banner**: Header image for the portal

#### Additional Information
- **NPSN**: National School Identification Number
- **NSS**: School Statistical Number
- **Accreditation**: Accreditation status and grade
- **Established Year**: Year of school foundation

---

### Action Buttons

- **Save**: Save all changes
- **Reset**: Revert to previous data
- **Preview**: View profile layout

---

### Important Notes

- Ensure all information is accurate and up to date
- Logos and images should be high resolution
- Profile changes will be visible throughout the system
