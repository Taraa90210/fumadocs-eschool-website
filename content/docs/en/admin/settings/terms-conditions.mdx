# Terms & Conditions

Page for managing the general terms and conditions for using the eSchool system.

---

### Main Function

Manage terms and conditions content that must be agreed to by all system users (students, parents, teachers, staff).

---

### Terms & Conditions Content

#### General Provisions
- Definition of terms
- Scope of services
- User rights and obligations
- Limitation of liability

#### Account Use
- Account creation and security
- Password confidentiality
- User responsibility for account
- Account suspension and termination

#### Privacy and Data
- Data collection
- Data use
- Data security
- User rights over data

#### Content and Behavior
- Content use rules
- Prohibited behavior
- Intellectual property rights
- Violation reporting

---

### Content Editor

- **Rich Text Editor**: Professional text format
- **Sections**: Divide into structured sections
- **Preview**: View layout for users
- **Version Control**: Manage different versions

---

### Approval Settings

- **Mandatory Approval**: Upon registration or first login
- **Update Notification**: Notification if there are changes
- **Approval Log**: Record time and approved version

---

### Usage Tips

- Use clear and unambiguous language
- Consult with legal experts
- Update regularly in accordance with regulations
- Notify users of important changes
