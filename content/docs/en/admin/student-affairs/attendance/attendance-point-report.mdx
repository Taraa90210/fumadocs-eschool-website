The Attendance Point Report page displays a recapitulation of student absence points in decimal units (hours) per semester, divided based on absence categories. This report functions as an audit tool and final semester data recapitulation.

---

### Report Filter

To display the attendance point report, you must specify the following filters:
- **Level**: Select the student class level (for example, X, XI, or XII).
- **Class**: Select the specific class you want to review.
- **Semester**: Select the semester to be reported (for example, Odd or Even).

After selecting all criteria, click the red **Generate** button to load the report.

---

### Attendance Sheet Format (Attendance Point Report)

This report table presents data in two formats: decimal points (actual) and rounding points (round).

| Main Column | Description |
| :--- | :--- |
| **NO, NISN, NAME, M/F** | Student identity data. |
| **SICK** | Total accumulated absence points due to illness (in decimal units, e.g., `0.3`). |
| **PERMISSION** | Total accumulated absence points due to permission (in decimal units). |
| **NO EXPLANATION** | Total accumulated absence points without explanation (in decimal units). |
| **SICK (ROUND)** | Total accumulated sick absence after being rounded to the nearest unit (usually in units of days). |
| **PERMISSION (ROUND)** | Total accumulated permission absence after being rounded to the nearest unit. |
| **NO EXPLANATION (ROUND)** | Total accumulated absence without explanation after being rounded to the nearest unit. |

> [!IMPORTANT]
> Attendance points are calculated based on lesson hours. For example, if 1 lesson hour is equivalent to `0.1` point, then 3 hours of absence will be recorded as `0.3` points.

---

### Report Usage

This report is **Read-Only**. You cannot change attendance point data from this page. Changes to attendance data must be made through the **Attendance Report** page.

You can use the action buttons on the top right of the report to:
- **Print**: Print the report into physical format.
- **Download**: Download the report into a file (for example, PDF or Excel) for archive purposes.
