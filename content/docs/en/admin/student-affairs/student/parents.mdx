# Parents

Page to manage parent or student guardian data.

---

### Edit Parent Data

To edit parent data:

1. Click the three-dot icon (â‹®) in the action column.
2. Select the **Edit** option.
3. An edit form will appear as shown in the image.

---

### Edit Parent Form

| Field | Description |
|-------|-------------|
| **First Name** | Parent's first name (required). |
| **Last Name** | Parent's last name (if any). |
| **Email** | Parent account email used to log into the system. |
| **Phone Number** | Active phone number for communication with the school. |
| **Gender** | Options: Male or Female. |
| **Upload Photo** | Optional, to add parent profile photo. |
| **Reset Password** | Check if you want to reset the parent account password. |

Click the **Save** button to save parent data changes.

---

### Reset Password Feature

If a parent forgets their account password:
- Admin can check the **Reset Password** option in this form.
- The system will reset the password and the parent can log in again with the default set by the system (usually announced by the school).

---

### Important Notes

- Ensure parent email is not duplicate (unique) to avoid account conflicts.
- If one parent has more than one child, the system will automatically connect them through the same account.
- Data changes here will directly impact the display in the parent portal.

---

### Practical Tips

- Use complete parent naming to avoid confusion.
- Always ensure Phone Number and Email are active so important information from the school can be received.
- It is recommended to upload parent photo for visual identification purposes in the school system.
