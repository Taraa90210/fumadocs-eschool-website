The **Student List** page is the control center for viewing, filtering, and managing data of all students within the educational unit. This feature is very useful for admins, school operators, and student affairs officers.

---

### Student List View

The student list table displays important information, including:

| Column | Description |
| :--- | :--- |
| **No** | Sequence number of the student display on the page |
| **Registration No** | Student registration number per academic year |
| **NIPD** | Student Identification Number (internal school) |
| **NISN** | National Student Identification Number |
| **Full Name** | Official student name |
| **Nickname** | Everyday or informal name |
| **Class** | Class currently occupied by the student |
| **Gender** | Male or Female |
| **Action** | Three dots icon to access additional features (view/edit data, delete, etc.) |

---

### Filter and Navigation Features

- **Academic Year, Level, and Class** can be filtered using dropdowns to adjust the data view.
- **Search** allows searching by name, NISN, or other info.
- **Active / Inactive Status** can be clicked in the top right corner to filter student display based on status.

### Save Changes

After all data is updated, click the **Save** button to save changes to the system.

---

### Tips & Notes

- Ensure no required columns are empty to avoid errors.
- Changes in data such as class, name, or NISN will be immediately updated in other integrated systems such as **attendance, grades, and login accounts**.
- Use this feature carefully as it involves the overall integrity of student data.
