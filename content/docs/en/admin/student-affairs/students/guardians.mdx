The **Guardian List** page displays all guardian data connected to active students at school. This data is important for communication needs, verification, and providing access to the guardian portal.

---

### Guardian List Table Display

This table displays important information such as:

| Column | Description |
|--------|-------------|
| **No** | Display sequence number |
| **First Name** | Guardian's first name |
| **Last Name** | Guardian's last name |
| **Gender** | Male / Female |
| **Email** | Guardian account email used for login |
| **Phone Number** | Guardian's phone number |
| **Action** | Three-dot icon (⋮) for editing data |

A search feature is also available to find guardians based on name, email, or phone number.

---

### Edit Guardian Data

To edit guardian data:

1. Click the **three-dot icon (⋮)** in the action column.
2. Select the **Edit** option.
3. An edit form will appear as shown in the image.

<img src="/images/edit-data-walimurid.png" alt="Description image" />

---

### Edit Guardian Form

| Field | Explanation |
|-------|-------------|
| **First Name** | Guardian's first name (required). |
| **Last Name** | Guardian's last name (if any). |
| **Email** | Guardian account email used to login to the system. |
| **Phone Number** | Active phone number for communication with the school. |
| **Gender** | Options: Male or Female. |
| **Upload Image** | Optional, to add guardian profile photo. |
| **Reset Password** | Check if you want to reset guardian account password. |

Click the **Save** button to save guardian data changes.

---

### Reset Password Feature

If guardians forget their account password:

- Admin can check the **Reset Password** option in this form.
- The system will reset the password and guardians can login again with the default set by the system (usually announced by the school).

---

### Important Notes

- Make sure guardian email is not duplicate (unique) to avoid account conflicts.
- If one guardian has more than one child, the system will automatically connect them through the same account.
- Changes here will directly impact the display in the guardian portal.

---

### Practical Tips

- Use complete guardian naming to avoid confusion.
- Always ensure Phone Number and Email are active so important information from school can be received.
- It is recommended to upload guardian photos for visual identification purposes in the school system.
