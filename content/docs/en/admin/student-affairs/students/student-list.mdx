The **Student List** page is a control center for viewing, filtering, and managing data of all students in the educational institution. This feature is very useful for admins, school operators, and student affairs officers.

---

### Student List Display

The student list table displays important information, including:

| Column | Description |
|--------|-------------|
| **No** | Student display sequence number on the page |
| **Registration No** | Student registration number per academic year |
| **NIPD** | Student Identification Number (internal school) |
| **NISN** | National Student Identification Number |
| **Full Name** | Student's official name |
| **Nickname** | Daily or informal name |
| **Class** | Class currently occupied by student |
| **Gender** | Male or Female |
| **Action** | Three-dot icon to access additional features (view/edit data, delete, etc) |

---

### Filter and Navigation Features

- **Academic Year, Level, and Class** can be filtered using dropdowns to adjust data display.
- **Search** allows searching based on name, NISN, or other info.
- **Active / Inactive Status** can be clicked in the upper right corner to filter student display based on status.

### Save Changes

After all data is updated, click the **Save** button to save changes to the system.

---

### Tips & Notes

- Make sure there are no empty required fields to avoid errors.
- Data changes such as class, name, or NISN will be directly updated in other integrated systems such as **attendance, grades, and login accounts**.
- Use this feature carefully as it concerns the integrity of student data as a whole.
