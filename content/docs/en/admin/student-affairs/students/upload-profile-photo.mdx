The **Upload Profile Photo** feature is used to update student and guardian identity photos quickly and in bulk based on class. This feature is very useful during administrative processes, student card printing, and visual data management.

---

### Access Menu

1. Go to **Student Affairs** > **Students** > **Upload Profile Photo** menu.
2. Select the class you want to display in the dropdown.
3. The list of students and guardians from that class will appear automatically.

---

### Table Display

After selecting a class, a table will appear displaying:

| Column | Description |
|--------|-------------|
| **No** | Student sequence number |
| **Name** | Student's full name |
| **Image** | Status of whether student already has a photo or not (`-` if not yet) |
| **Guardian** | Name of registered and connected guardian |
| **Guardian Image** | Photo icon if guardian already has an image |
| **Student Profile** | Student image upload form (Choose File) |
| **Guardian Profile** | Guardian image upload form (Choose File) |

---

### How to Upload Photos

1. Click the **Choose File** button in the Student Profile or Guardian Profile column as needed.
2. Select an image file from your computer. Supported formats generally: `.jpg`, `.jpeg`, `.png`.
3. After all files are selected, the system will automatically save or there is an additional button like **Save** (depending on system version).
4. The image will appear replacing the default icon after successfully uploaded.

> [!TIP]
> Use consistent file names to make them easy to find and recognize if problems occur.

---

### Technical Notes

- File size is recommended maximum **500KB** to avoid burdening the system.
- File format must be an image (not PDF or document).
- The system will automatically link the image to the student or guardian account based on class and data ID.

---

### When to Use?

- When new students enter and don't have photos yet.
- Preparing digital or physical student cards.
- For re-data collection or updating student & guardian identities.
- When there are data changes or update requests from parents.
