# Categories

The Expense Categories page is used to define and manage the types of categories that will be used when recording expense transactions. With categories, financial recording becomes more structured and easy to analyze.

---

### Add Expense Category

This section is used to create a new expense category. Make sure you fill in all required columns (marked with an asterisk).

| Field | Description |
|-------|-------------|
| **Name** | Expense category name (for example, Library, Exam Activities, Building Maintenance). |
| **Description** | Brief explanation regarding the types of expenses included in this category. |

After filling in the data, click the red **Save** button to save the new category.

---

### Expense Category List

This section displays all expense categories that have been recorded in the system.

#### Filter and Search

You can filter category data through:
- **Search**: Use the **Search** column to search for categories based on name or description.
- **Quick Buttons**:
  - Click the **All** button to display the entire list of active categories.
  - Click the **Deleted** button to see the list of categories that have been temporarily deleted (soft delete).

#### Expense Category Table

| Column | Description |
|--------|-------------|
| **No** | Category data sequence number. |
| **Name** | Expense category name. |
| **Description** | Detailed explanation regarding that category. |
| **Action** | Button to manage category data. |

---

### Action (Data Management)

In the **Action** column, there is a vertical menu button that provides options to manage categories:

#### 1. Active Categories (When Accessing "All" View)
When you view the list of active categories, the options available in the Action button are:
- **Edit**: Option to change the name and description of the selected category.
- **Delete**: Option to delete the category temporarily (soft delete). Deleted categories will move to the "Deleted" view.

#### 2. Deleted Categories (When Accessing "Deleted" View)
When you click the **Deleted** button and view the list of temporarily deleted categories, the options available in the Action button will change to:
- **Restore**: Option to return the temporarily deleted category so it reappears in the active category list ("All").
- **Permanent Delete**: Option to permanently delete category data from the system, and data cannot be recovered.
