# Expenses

The Expenses page functions to record and manage all school financial expense transactions. This page is divided into two main parts: a form to add new expenses and a table to display the list of expenses that have been recorded.

---

### Add Expense

This section is used to enter new expense data into the system. Required columns (marked with an asterisk) must be filled.

| Column | Description |
|--------|-------------|
| **Select Category** | Expense category (for example, operational, inventory, salary). |
| **Title** | Name or brief topic of that expense. |
| **Reference No** | Unique number or reference code for this transaction (optional). |
| **Amount (Rp)** | Nominal money spent in Rupiah. |
| **Date** | Date the expense occurred. |
| **Select Academic Year** | Academic year where this expense is recorded (for example, 2025/2026). |
| **Description** | Details or complete explanation regarding that expense. |

After all data is filled in, click the red **Save** button to record the expense.

---

### Expense List

This table displays all expense records that have been entered.

#### Filter and Search

You can filter expense data based on:
- **Academic Year**: Filter based on a specific academic year.
- **Category**: Filter based on expense category.
- **Month**: Filter based on the month the expense occurred.
- **Search**: Use the search column to search for data based on keywords.

At the bottom of the table, there is **Total (Rp)** information showing the total accumulation of all expenses currently displayed.

#### Expense Table

| Column | Description |
|--------|-------------|
| **No** | Expense data sequence number. |
| **Reference No** | Transaction reference number (if any). |
| **Title** | Title or expense name. |
| **Date** | Date the expense occurred. |
| **Category** | Expense category. |
| **Amount (Rp)** | Nominal money spent in Rupiah. |
| **Action** | Button to manage this expense data. |

---

### Action (Data Management)

In the **Action** column, you have buttons that can be used to manage existing expense data:
- **Edit Expense**: Click the gray 'i' (information) icon button in the Action column to open the form and make changes to that expense data.
- **Delete Expense**: The option to delete expenses is usually available in the detail or edit form after you click the action button. Find and click the delete button to permanently delete the expense record.
