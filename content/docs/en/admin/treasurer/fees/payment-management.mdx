# Payment Management

Page for managing and processing payments from students.

---

### Main Functions

The Payment Management page is used to:
- Record payments made by students
- Process payment transactions
- Issue payment receipts
- Update student billing status

---

### How to Use

1. Select the student who will make payment
2. Select the fee type to be paid
3. Enter payment nominal
4. Select payment method (cash, transfer, etc.)
5. Click **Save** to process payment

---

### Additional Features

- **Print Receipt**: Print payment receipt for students
- **Payment History**: View student payment history
- **Partial Payment**: Allow installment payments
