# **Main Dashboard Functions**
The Dashboard page presents a summary of important information in real-time, such as the number of teachers, active students, inactive students, student guardians, classes, and majors.

This display makes it easy for schools to monitor current activities and support fast and accurate decision making.

# **Form Display (Form Input/Edit)**

Used to add or change data, such as student registration, add teachers, financial data, and academic settings.



| Feature / Button   | Function                                                    |
| :--------------- | :-------------------------------------------------------- |
| **Input Group** | Group inputs based on categories (personal data, academic, etc)         |
| **Save** | Save new data or data changes |
| **Delete/Reset**      | Delete old data or clear form                 |
| **Automatic Validation**  | System will mark if there is incomplete/invalid input |
