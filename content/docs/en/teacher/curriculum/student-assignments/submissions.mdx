The **Submissions** page within the **Student Assignments** module of eSchool is a centralized control center for teachers to monitor and evaluate all student work. Through this interface, teachers can view the list of assignments submitted by students, track grading status, and provide direct scores and feedback.

### üßæ **Submission Evaluation Form**

<img src="/images/form-penyerahan.png" alt="Image description" className="mx-auto block max-w-full" />

Complete the form based on the following instructions:

| Field | Explanation |
| :--- | :--- |
| **Assignment Name** | The title of the assignment (inherited and read-only). |
| **Subject** | The related academic subject (automatically displayed). |
| **Student Name** | The full name of the student. |
| **Text Response** | Displays direct text entries if the student submitted an answer without a file. |
| **File** | Displays uploaded attachments; click the icon to view content directly. |
| **Status** *(Required)* | Select either `Accepted` or `Rejected`. |
| **Points** *(Optional)* | Enter the final numerical score obtained (e.g., 85). |
| **Feedback** *(Optional)* | Provide notes or comments to guide the student. |

‚úÖ Once completed, click the **Save** button to persist the evaluation.
