The **Announcements** feature in the eSchool platform is designed to make it easier for teachers, education personnel, and school admins to disseminate important information to students or specific classes. Announcements are displayed in a table format and can be accompanied by supporting files or documents.

### ‚ûï How to Add an Announcement

To create a new announcement, follow these steps:

#### 1. Fill in the Add Announcement Form:

* **Title** *(Required)*: Enter the announcement title, e.g., "Islamic New Year Holiday".
* **Description**: Write the content or a brief description of the announcement.
* **File (Optional)**: Click the **"Upload"** button to add supporting images or documents (max. 3 files).
* **Class Section**: Select the class that will receive the announcement.
* **Subject (Optional)**: Select if the announcement relates to a specific subject.

#### 2. Click **Save** to save the announcement to the list.

To view the full content of an announcement:

* **Directly click on the title text** in the announcement list table.
* A detail window will appear containing:
  * Full title and content of the announcement.
  * Date description & recipient.
  * Attachments if any (files can be downloaded via the **View** button).
