The **Holidays** feature in the eSchool platform is designed to assist schools in documenting and delivering official information regarding holiday schedules to school members. Whether it's a national holiday, internal school activity, or academic leave, everything is presented in a neat and easily filterable list.

### ðŸ” Search Filter

To filter the displayed holiday data, use the following filters:

* **Academic Year**: Specify the academic period, e.g., `2025/2026`.
* **Month**: Select a specific month to see relevant holidays.
* **Start Date and End Date**: Narrow down results based on date range.
* **Search Bar**: Search based on a specific title or description.

***

### ðŸ“‹ Holiday List

Each entry in the table displays information as follows:

| Column | Description |
| :--- | :--- |
| **No** | Sequence number |
| **Start Date** | Holiday start date |
| **End Date** | Holiday end date |
| **Title** | Name of the holiday activity, such as "Independence Day Preparation Holiday" |
| **Description** | Brief summary of the explanation related to the holiday |

***

### ðŸ‘ï¸ Viewing Holiday Details in a Pop-up

> ðŸ”” **User Guide:**
> To view the full details of each holiday, **click directly on the description text** in the table column.

After clicking, a **modal window (pop-up)** will appear displaying complete information, including:

* Announcement title
* Full narrative content of the announcement
* Important dates to note
* Person in charge or official source (if any)

Example: When a user clicks the brief description "ANNOUNCEMENT Number: 045/...", a modal will appear with the official announcement content about **UAS Preparation Holiday**, including exam dates and academic notes.
