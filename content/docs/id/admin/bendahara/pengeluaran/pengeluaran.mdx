Halaman Pengeluaran berfungsi untuk mencatat dan mengelola semua transaksi pengeluaran keuangan sekolah. Halaman ini terbagi menjadi dua bagian utama: formulir untuk menambahkan pengeluaran baru dan tabel untuk menampilkan daftar pengeluaran yang sudah tercatat.

---

### Tambah Pengeluaran

Bagian ini digunakan untuk memasukkan data pengeluaran baru ke dalam sistem. Kolom yang wajib diisi (ditandai dengan bintang) harus dipenuhi.

| Kolom | Deskripsi |
|-------|-----------|
| **Pilih Kategori** | Kategori pengeluaran (misalnya, operasional, inventaris, gaji). |
| **Judul** | Nama atau topik singkat dari pengeluaran tersebut. |
| **No Referensi** | Nomor unik atau kode referensi untuk transaksi ini (opsional). |
| **Jumlah (Rp)** | Nominal uang yang dikeluarkan dalam Rupiah. |
| **Tanggal** | Tanggal terjadinya pengeluaran. |
| **Pilih Tahun Ajaran** | Tahun ajaran di mana pengeluaran ini dicatat (misalnya, 2025/2026). |
| **Keterangan** | Detail atau penjelasan lengkap mengenai pengeluaran tersebut. |

Setelah semua data terisi, klik tombol **Simpan** berwarna merah untuk mencatat pengeluaran.

---

### Daftar Pengeluaran

Tabel ini menampilkan seluruh catatan pengeluaran yang telah dimasukkan.

#### Filter dan Pencarian

Anda dapat menyaring data pengeluaran berdasarkan:
- **Tahun Ajaran**: Filter berdasarkan tahun ajaran tertentu.
- **Kategori**: Filter berdasarkan kategori pengeluaran.
- **Bulan**: Filter berdasarkan bulan terjadinya pengeluaran.
- **Search**: Gunakan kolom pencarian untuk mencari data berdasarkan kata kunci.

Di bawah tabel, terdapat informasi **Total (Rp)** yang menunjukkan total akumulasi dari semua pengeluaran yang ditampilkan saat ini.

#### Tabel Pengeluaran

| Kolom | Deskripsi |
|-------|-----------|
| **No** | Nomor urut data pengeluaran. |
| **No Referensi** | Nomor referensi transaksi (jika ada). |
| **Judul** | Judul atau nama pengeluaran. |
| **Tanggal** | Tanggal terjadinya pengeluaran. |
| **Kategori** | Kategori pengeluaran. |
| **Jumlah (Rp)** | Nominal uang yang dikeluarkan dalam Rupiah. |
| **Aksi** | Tombol untuk mengelola data pengeluaran ini. |

---

### Aksi (Pengelolaan Data)

Pada kolom **Aksi**, Anda memiliki tombol yang dapat digunakan untuk mengelola data pengeluaran yang sudah ada:
- **Edit Pengeluaran**: Klik tombol gambar ikon 'i' (informasi) berwarna abu-abu di kolom Aksi untuk membuka formulir dan melakukan perubahan pada data pengeluaran tersebut.
- **Hapus Pengeluaran**: Opsi untuk menghapus pengeluaran biasanya tersedia di dalam detail atau formulir edit setelah Anda mengklik tombol aksi. Cari dan klik tombol hapus untuk menghapus catatan pengeluaran secara permanen.
