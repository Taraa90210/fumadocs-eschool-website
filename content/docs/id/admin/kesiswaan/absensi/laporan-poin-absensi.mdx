Halaman Laporan Poin Absensi menampilkan rekapitulasi poin ketidakhadiran siswa dalam satuan desimal (jam) per semester, yang terbagi berdasarkan kategori ketidakhadiran. Laporan ini berfungsi sebagai alat audit dan rekapitulasi data akhir semester.

---

### Filter Laporan

Untuk menampilkan laporan poin absensi, Anda harus menentukan filter berikut:
- **Tingkatan**: Pilih tingkatan kelas siswa (misalnya, X, XI, atau XII).
- **Kelas**: Pilih kelas spesifik yang ingin Anda tinjau.
- **Semester**: Pilih semester yang ingin dilaporkan (misalnya, Ganjil atau Genap).

Setelah memilih semua kriteria, klik tombol **Generate** berwarna merah untuk memuat laporan.

---

### Format Lembar Absensi (Laporan Poin Absensi)

Tabel laporan ini menyajikan data dalam dua format: poin desimal (aktual) dan poin pembulatan (round).

| Kolom Utama | Deskripsi |
|-------------|-----------|
| **NO, NISN, NAMA, L/P** | Data identitas siswa. |
| **SAKIT** | Total akumulasi poin ketidakhadiran karena sakit (dalam satuan desimal, misal `0.3`). |
| **IJIN** | Total akumulasi poin ketidakhadiran karena izin (dalam satuan desimal). |
| **TANPA KETERANGAN** | Total akumulasi poin ketidakhadiran tanpa keterangan (dalam satuan desimal). |
| **SAKIT (ROUND)** | Total akumulasi ketidakhadiran sakit setelah dibulatkan ke satuan terdekat (biasanya dalam satuan hari). |
| **IJIN (ROUND)** | Total akumulasi ketidakhadiran izin setelah dibulatkan ke satuan terdekat. |
| **TANPA KETERANGAN (ROUND)** | Total akumulasi ketidakhadiran tanpa keterangan setelah dibulatkan ke satuan terdekat. |

> [!IMPORTANT]
> Poin ketidakhadiran dihitung berdasarkan jam pelajaran. Misalnya, jika 1 jam pelajaran setara dengan `0.1` poin, maka ketidakhadiran 3 jam akan dicatat sebagai `0.3` poin.

---

### Penggunaan Laporan

Laporan ini bersifat **Read-Only (Hanya Baca)**. Anda tidak dapat mengubah data poin absensi dari halaman ini. Perubahan data absensi harus dilakukan melalui halaman **Laporan Absensi**.

Anda dapat menggunakan tombol aksi di sisi kanan atas laporan untuk:
- **Mencetak**: Mencetak laporan ke dalam format fisik.
- **Mengunduh**: Mengunduh laporan ke dalam file (misalnya, PDF atau Excel) untuk keperluan arsip.
