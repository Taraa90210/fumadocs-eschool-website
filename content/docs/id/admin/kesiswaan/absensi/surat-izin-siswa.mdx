Halaman Surat Izin Siswa adalah pusat pengelolaan dan persetujuan permohonan izin tidak masuk sekolah yang diajukan oleh siswa atau wali siswa. Anda dapat melihat, memfilter, dan memproses status perizinan dari halaman ini.

---

### Filter Data Perizinan

Anda dapat memfilter daftar surat izin berdasarkan kriteria berikut:
- **Kelas**: Saring data perizinan berdasarkan kelas siswa (misalnya, X, XI, XII).
- **Tgl Mulai & Tgl Selesai**: Tentukan rentang tanggal perizinan yang ingin Anda tampilkan.
- **Search**: Gunakan kolom pencarian untuk menemukan data berdasarkan nama siswa atau kata kunci lainnya.

---

### Daftar Surat Izin Siswa

Tabel ini menampilkan ringkasan dari setiap permohonan izin yang masuk.

| Kolom | Deskripsi |
|-------|-----------|
| **No** | Nomor urut permohonan. |
| **Nama** | Nama lengkap siswa yang mengajukan izin. |
| **Kelas** | Kelas siswa. |
| **Alasan** | Ringkasan alasan perizinan yang diajukan. |
| **Tanggal** | Tanggal perizinan (tanggal tidak masuk sekolah). |
| **Status** | Status terkini dari perizinan (misalnya, Tertunda, Disetujui, Ditolak). |
| **Aksi** | Tombol untuk memproses status perizinan. |

---

### Pemrosesan Status Perizinan (Kolom Aksi)

Kolom Aksi menyediakan dua tombol untuk langsung memproses perizinan:
- **Menyetujui Perizinan**: Klik tombol centang berwarna hijau untuk langsung mengubah status perizinan menjadi **Disetujui**.
- **Menolak Perizinan**: Klik tombol silang berwarna merah muda untuk langsung mengubah status perizinan menjadi **Ditolak**.

Anda juga dapat memproses beberapa perizinan sekaligus dengan mencentang kotak di samping nomor urut dan mengklik tombol **Izinkan** di atas tabel.

### Detail Perizinan Siswa

Untuk melihat informasi lengkap mengenai suatu permohonan izin, klik salah satu baris data yang ada di tabel. Sebuah modal (jendela pop-up) **Detail Perizinan Siswa** akan muncul.

#### Bagian Data Diri Siswa

Bagian ini menampilkan informasi dasar siswa dan detail perizinan:
- Nama Siswa dan NISN.
- Kelas dan No Absen.
- Tanggal Perizinan dan Status perizinan (misalnya, Tertunda).
- **Alasan Perizinan**: Alasan lengkap yang diajukan.
- **Alasan Penolakan**: Kolom ini akan terisi jika Anda memilih untuk menolak perizinan, memberikan catatan mengapa permohonan ditolak.

#### Bagian Lampiran Foto

Bagian ini menampilkan gambar yang diunggah oleh siswa/wali sebagai bukti atau lampiran pendukung permohonan izin. Anda dapat mengklik gambar untuk memperbesarnya.

---

### Pemrosesan Detail

Di bagian bawah modal detail, terdapat dua tombol untuk menentukan status perizinan:
- **Setujui**: Klik tombol **Setujui** berwarna hijau untuk menyetujui permohonan izin.
- **Tolak**: Klik tombol **Tolak** berwarna merah muda untuk menolak permohonan izin. Jika menolak, pastikan Anda mengisi kolom **Alasan Penolakan** pada bagian Data Diri Siswa.
