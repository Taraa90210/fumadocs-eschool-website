Halaman **Daftar Siswa** adalah pusat kontrol untuk melihat, menyaring, dan mengelola data seluruh peserta didik dalam satuan pendidikan. Fitur ini sangat berguna bagi admin, operator sekolah, dan petugas kesiswaan.

---

### Tampilan Daftar Siswa

Tabel daftar siswa menampilkan informasi penting, meliputi:

| Kolom | Keterangan |
|-------|------------|
| **No** | Nomor urutan tampilan siswa di halaman |
| **No Pendaftaran** | Nomor registrasi siswa per tahun ajaran |
| **NIPD** | Nomor Induk Peserta Didik (internal sekolah) |
| **NISN** | Nomor Induk Siswa Nasional |
| **Nama Lengkap** | Nama resmi siswa |
| **Nama Panggilan** | Nama sehari-hari atau informal |
| **Kelas** | Kelas yang sedang ditempati siswa |
| **Jenis Kelamin** | Laki-laki atau Perempuan |
| **Aksi** | Ikon titik tiga untuk mengakses fitur tambahan (lihat/edit data, hapus, dll) |

---

### Fitur Filter dan Navigasi

- **Tahun Ajaran, Tingkatan, dan Kelas** dapat disaring menggunakan dropdown untuk menyesuaikan tampilan data.
- **Cari** memungkinkan pencarian berdasarkan nama, NISN, atau info lainnya.
- **Status Aktif / Nonaktif** dapat diklik di pojok kanan atas untuk memfilter tampilan siswa berdasarkan status.

### Simpan Perubahan

Setelah semua data diperbarui, klik tombol **Simpan** untuk menyimpan perubahan ke sistem.

---

### Tips & Catatan

- Pastikan tidak ada kolom wajib yang kosong untuk menghindari error.
- Perubahan data seperti kelas, nama, atau NISN akan langsung terupdate di sistem lain yang terintegrasi seperti **absensi, nilai, dan akun login**.
- Gunakan fitur ini dengan hati-hati karena menyangkut integritas data siswa secara menyeluruh.
