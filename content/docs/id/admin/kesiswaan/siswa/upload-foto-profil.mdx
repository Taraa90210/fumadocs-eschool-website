Fitur **Upload Foto Profil** digunakan untuk memperbarui foto identitas siswa dan wali murid secara cepat dan massal berdasarkan kelas. Fitur ini sangat berguna saat proses administrasi, pencetakan kartu pelajar, dan pengelolaan data visual.

---

### Akses Menu

1. Masuk ke menu **Kesiswaan** > **Siswa** > **Upload Foto Profil**.
2. Pilih kelas yang ingin ditampilkan pada dropdown.
3. Daftar siswa dan wali dari kelas tersebut akan muncul secara otomatis.

---

### Tampilan Tabel

Setelah memilih kelas, akan muncul tabel yang menampilkan:

| Kolom | Keterangan |
|-------|------------|
| **No** | Nomor urut siswa |
| **Nama** | Nama lengkap siswa |
| **Gambar** | Status apakah siswa sudah memiliki foto atau belum (`-` jika belum) |
| **Wali** | Nama wali murid yang terdaftar dan terhubung |
| **Gambar Wali** | Ikon foto jika wali sudah memiliki gambar |
| **Profil Siswa** | Form upload gambar siswa (Choose File) |
| **Profil Wali** | Form upload gambar wali murid (Choose File) |

---

### Cara Mengunggah Foto

1. Klik tombol **Choose File** pada kolom Profil Siswa atau Profil Wali sesuai kebutuhan.
2. Pilih file gambar dari komputer Anda. Format yang didukung umumnya: `.jpg`, `.jpeg`, `.png`.
3. Setelah semua file dipilih, sistem akan otomatis menyimpan atau ada tombol tambahan seperti **Simpan** (tergantung versi sistem).
4. Gambar akan tampil menggantikan ikon default setelah berhasil diunggah.

> [!TIP]
> Gunakan nama file yang konsisten agar mudah dicari dan dikenali jika terjadi kendala.

---

### Catatan Teknis

- Ukuran file disarankan maksimal **500KB** agar tidak memberatkan sistem.
- Format file harus gambar (bukan PDF atau dokumen).
- Sistem akan otomatis menautkan gambar ke akun siswa atau wali murid berdasarkan kelas dan ID data.

---

### Kapan Digunakan?

- Saat siswa baru masuk dan belum ada foto.
- Menyusun kartu pelajar digital atau fisik.
- Untuk pendataan ulang atau memperbarui identitas siswa & wali murid.
- Ketika ada perubahan data atau permintaan update dari orang tua.
