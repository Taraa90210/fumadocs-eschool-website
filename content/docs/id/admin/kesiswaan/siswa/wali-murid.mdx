Halaman **Daftar Wali** menampilkan seluruh data wali murid yang terhubung dengan siswa aktif di sekolah. Data ini penting untuk kebutuhan komunikasi, verifikasi, dan pemberian akses portal wali murid.

---

### Tampilan Tabel Daftar Wali

Tabel ini menampilkan informasi penting seperti:

| Kolom | Keterangan |
|-------|------------|
| **No** | Nomor urutan tampilan |
| **Nama Depan** | Nama depan wali |
| **Nama Belakang** | Nama belakang wali |
| **Jenis Kelamin** | Laki-laki / Perempuan |
| **Email** | Email akun wali yang digunakan login |
| **No HP** | Nomor telepon wali murid |
| **Aksi** | Ikon titik tiga (⋮) untuk edit data |

Tersedia juga fitur pencarian (search) untuk menemukan wali berdasarkan nama, email, atau nomor HP.

---

### Edit Data Wali Murid

Untuk mengedit data wali murid:

1. Klik **ikon titik tiga (⋮)** di kolom aksi.
2. Pilih opsi **Edit**.
3. Akan muncul form edit seperti yang terlihat pada gambar.

<img src="/images/edit-data-walimurid.png" alt="Description image" />

---

### Formulir Edit Wali

| Field | Penjelasan |
|-------|------------|
| **Nama Depan** | Nama depan wali murid (wajib). |
| **Nama Belakang** | Nama belakang wali (jika ada). |
| **Email** | Email akun wali yang digunakan untuk login ke sistem. |
| **No HP** | Nomor telepon aktif untuk komunikasi dengan pihak sekolah. |
| **Jenis Kelamin** | Pilihan: Laki-laki atau Perempuan. |
| **Unggah Gambar** | Opsional, untuk menambahkan foto profil wali murid. |
| **Reset Password** | Centang jika ingin mengatur ulang password akun wali. |

Klik tombol **Simpan** untuk menyimpan perubahan data wali.

---

### Fitur Reset Password

Jika wali murid lupa password akun mereka:

- Admin dapat mencentang opsi **Reset Password** di form ini.
- Sistem akan mengatur ulang password dan wali bisa login kembali dengan default yang telah diatur oleh sistem (biasanya diumumkan oleh sekolah).

---

### Catatan Penting

- Pastikan email wali murid tidak ganda (unik) agar tidak terjadi konflik akun.
- Jika satu wali memiliki lebih dari satu anak, sistem secara otomatis akan menghubungkan mereka melalui akun yang sama.
- Perubahan data di sini akan langsung berdampak pada tampilan di portal wali murid.

---

### Tips Praktis

- Gunakan penamaan wali secara lengkap agar tidak rancu.
- Selalu pastikan No HP dan Email aktif agar informasi penting dari sekolah dapat diterima.
- Disarankan mengunggah foto wali untuk keperluan identifikasi visual di sistem sekolah.
