Halaman Edit Soal digunakan untuk mengubah isi dari soal yang sudah dibuat dalam Bank Soal. Fitur ini penting untuk memperbarui atau menyempurnakan materi ujian tanpa harus membuat soal baru dari nol.

---

### Cara Mengakses

1. Arahkan ke menu: **Kurikulum** > **Ujian Online** > **Bank Soal**.
2. Pilih mata pelajaran dan daftar soal yang ingin diedit.
3. Pada daftar soal, klik ikon aksi (â‹®) di samping soal.
4. Pilih opsi **Edit**, maka halaman ini akan terbuka.

---

### Komponen di Halaman Edit Soal

| Komponen | Deskripsi |
|----------|-----------|
| **Versi Soal** | Menunjukkan versi revisi soal. Biasanya otomatis terisi. |
| **Nama Soal** | Judul soal untuk identifikasi internal (tidak muncul ke siswa). |
| **Tipe Soal** | Jenis soal yang dipilih, seperti Benar-Salah, Pilihan Ganda, Numerik, Esai, atau Jawaban Singkat. |
| **Poin Bawaan** | Nilai default jika siswa menjawab dengan benar. |
| **Gambar** | Bisa menambahkan ilustrasi atau gambar penunjang soal. Klik "Unggah" untuk mengganti gambar lama. |
| **Teks Soal** | Isi utama dari soal yang akan ditampilkan ke siswa. Bisa berupa teks panjang atau pendek. |
| **Catatan (Opsional)** | Catatan tambahan untuk admin/guru, tidak ditampilkan ke siswa. |
| **Opsi Jawaban** | Tempat untuk mengatur jawaban benar/salah atau pilihan lainnya tergantung tipe soal. Setiap opsi punya poin dan label. |


---

### Contoh Kasus

Misalnya kamu sedang merevisi soal Benar-Salah seperti:

*"Sit-up adalah latihan kebugaran yang bertujuan untuk melatih otot perut."*

Kamu bisa:
- Memperbaiki redaksi kalimat jika kurang jelas.
- Menambahkan gambar ilustrasi gerakan sit-up.
- Mengubah opsi jawaban kalau ternyata ada kekeliruan.
- Mengubah poin soal jika nilainya perlu disesuaikan.

---

### Langkah Terakhir

Setelah semua perubahan dilakukan, klik tombol **Simpan** untuk menyimpan hasil editan.
