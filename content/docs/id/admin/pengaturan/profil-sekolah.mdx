Halaman untuk mengelola informasi profil dan identitas sekolah.

---

### Tujuan

Menyediakan antarmuka untuk mengatur dan memperbarui informasi dasar sekolah yang ditampilkan di seluruh sistem.

---

### Komponen dan Pengaturan

#### Informasi Dasar
- **Nama Sekolah**: Nama resmi institusi
- **Alamat**: Alamat lengkap sekolah
- **Nomor Telepon**: Kontak sekolah
- **Email**: Email resmi sekolah
- **Website**: URL website sekolah

#### Identitas Visual
- **Logo Sekolah**: Upload logo resmi (format: jpg, png, svg)
- **Favicon**: Icon untuk browser tab
- **Banner**: Gambar header untuk portal

#### Informasi Tambahan
- **NPSN**: Nomor Pokok Sekolah Nasional
- **NSS**: Nomor Statistik Sekolah
- **Akreditasi**: Status dan nilai akreditasi
- **Tahun Berdiri**: Tahun pendirian sekolah

---

### Tombol Aksi

- **Simpan**: Menyimpan semua perubahan
- **Reset**: Mengembalikan ke data sebelumnya
- **Preview**: Melihat tampilan profil

---

### Catatan Penting

- Pastikan semua informasi akurat dan terkini
- Logo dan gambar sebaiknya beresolusi tinggi
- Perubahan profil akan terlihat di seluruh sistem
