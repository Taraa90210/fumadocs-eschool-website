Halaman **Impor Data Staf** digunakan untuk menambahkan data pegawai secara massal ke dalam sistem **E-SCHOOL SIAKAD Plus** melalui unggahan berkas berformat **.CSV**. Fitur ini memudahkan administrator dalam menginput banyak data staf sekaligus, tanpa harus menambahkan satu per satu secara manual.

***

### **Tujuan**

Fitur ini bertujuan untuk:

* Mempercepat proses input data staf.
* Meminimalisir kesalahan input manual.
* Menyediakan format standar untuk konsistensi data.

***

### **Langkah-langkah Impor Data**

<img src="/images/langkah-langkah-impor-data.png" alt="Image description" className="w-180 mx-auto"/>

1. **Pilih Peran**\
   Pilih peran staf yang sesuai pada kolom *Peran*. Peran ini akan berlaku untuk seluruh data staf yang diimpor.
2. **Unduh File Contoh**\
   Klik tombol **File Contoh** untuk mengunduh berkas CSV berisi format standar yang harus diikuti.\
   Struktur kolom pada file contoh adalah:

* `first_name` → Nama depan
* `last_name` → Nama belakang
* `mobile` → Nomor telepon
* `email` → Alamat email
* `dob` → Tanggal lahir (format: DD-MM-YYYY)
* `salary` → Gaji pokok
* `joining_date` → Tanggal bergabung (format: DD-MM-YYYY)

\
\&#xNAN;*(Contoh struktur dapat dilihat pada gambar tersebut)*

1. **Isi Data**\
   Lengkapi berkas CSV dengan data staf sesuai format. Pastikan tidak ada kolom yang terhapus atau diubah namanya.
2. **Unggah File**
   * Klik tombol **Unggah**, pilih berkas CSV yang sudah diisi.
   * Pastikan jenis file adalah **.CSV**.
3. **Simpan Data**\
   Setelah file terunggah, klik **Simpan** untuk memproses dan menambahkan data ke sistem.

***

### **Catatan Penting**

* Format tanggal wajib mengikuti format **DD-MM-YYYY** agar data terbaca dengan benar oleh sistem.
* Pastikan alamat email valid untuk menghindari kegagalan pengiriman informasi ke staf.
* Periksa kembali data sebelum diunggah untuk meminimalkan kesalahan.
* Sistem hanya menerima file dengan ekstensi **.CSV**.
