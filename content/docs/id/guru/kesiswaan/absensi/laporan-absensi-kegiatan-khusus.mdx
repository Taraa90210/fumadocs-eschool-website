Halaman **Laporan Absensi Kegiatan Khusus** di E-SCHOOL Siakad Plus dirancang untuk mencatat dan memantau kehadiran siswa dalam kegiatan **non-pelajaran.** Fitur ini memisahkan absensi kegiatan khusus dari absensi rutin harian dan per mata pelajaran, sehingga pelaporan menjadi lebih spesifik dan tertata.

<img src="/images/laporan-absensi-kegiatan-khusus.png" alt="Deskripsi gambar"  />

## Navigasi dan Filter

#### Filter Data

Pengguna dapat memfilter laporan berdasarkan:

* **Kelas**: Menampilkan laporan kegiatan khusus per kelas.
* **Tanggal Mulai & Tanggal Selesai**: Menentukan rentang waktu pencatatan kegiatan.
* **Jenis Absensi**: Menyaring data berdasarkan status kehadiran (Hadir, Sakit, Izin, Alpa).

Filter ini memudahkan pencarian data presensi kegiatan pada periode dan kelompok tertentu.

### Fitur Tambahan

* **Search Bar**: Memudahkan pencarian nama siswa secara cepat.
* **Ekspor Data**: Tersedia tombol untuk mengunduh laporan dalam format Excel atau PDF.
* **Pagination**: Navigasi antar halaman untuk memantau data lebih dari 10 entri per halaman.

***

### Manfaat Fitur Ini

| Pengguna      | Manfaat Khusus                                                         |
| ------------- | ---------------------------------------------------------------------- |
| Guru Kegiatan | Mendata peserta aktif kegiatan di luar kelas secara presisi            |
| Wali Kelas    | Mengetahui keaktifan siswa dalam program non-akademik                  |
| Kesiswaan     | Membuat laporan disiplin & keaktifan siswa lintas kegiatan sekolah     |
| Manajemen     | Mendapatkan insight kehadiran siswa dalam konteks kegiatan kelembagaan |

***

### Tips Penggunaan

* Gunakan filter **tanggal** untuk rekap kegiatan per event.
* Unduh laporan PDF untuk arsip dokumentasi sekolah.
* Pantau siswa yang sering tidak hadir dalam kegiatan sekolah untuk intervensi BK/kesiswaan.
