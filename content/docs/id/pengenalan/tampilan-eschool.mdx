# **Tampilan Formulir(Form Input/Edit)**

Digunakan untuk menambah atau mengubah data, seperti pendaftaran siswa, tambah guru, data keuangan, dan pengaturan akademik.



| Fitur / Tombol   | Fungsi                                                    |
| :--------------- | :-------------------------------------------------------- |
| **Grup Isian** | Mengelompokkan input berdasarkan kategori (data diri, akademik, dll)         |
| **Simpan** | Menyimpan data baru atau perubahan data |
| **Hapus/Reset**      | Menghapus data lama atau mengosongkan form                 |
| **Validasi Otomatis**  | Sistem akan memberi tanda jika ada input yang belum lengkap/valid |
