### Cara Reset Password

#### Metode 1: Self-Service Reset (Direkomendasikan)

1. **Buka Halaman Login**
   - Akses halaman login eSchool
   - Klik link "Lupa Password?" di bawah form login

2. **Masukkan Email/Username**
   - Ketik email atau username yang terdaftar
   - Klik tombol "Kirim Link Reset"

3. **Cek Email Anda**
   - Buka inbox email yang terdaftar
   - Cari email dari "noreply@eschool.ac.id"
   - **Catatan**: Periksa folder Spam/Junk jika tidak ada di Inbox

4. **Klik Link Reset**
   - Buka email dan klik link reset password
   - Link berlaku selama 1 jam
   - Jika expired, ulangi proses dari awal

5. **Buat Password Baru**
   - Masukkan password baru (minimal 8 karakter)
   - Konfirmasi password baru
   - Klik "Reset Password"

6. **Login dengan Password Baru**
   - Kembali ke halaman login
   - Gunakan password baru Anda

---

### Metode 2: Melalui Admin Sekolah

Jika metode self-service tidak berhasil:

1. **Hubungi Admin Sekolah**
   - Datang langsung ke ruang TU/IT
   - Atau hubungi via telepon/email sekolah

2. **Verifikasi Identitas**
   - Tunjukkan ID/kartu pelajar/kartu pegawai
   - Berikan informasi akun (username/email)

3. **Admin Reset Password**
   - Admin akan mereset password Anda
   - Anda akan menerima password sementara

4. **Login dan Ganti Password**
   - Login dengan password sementara
   - Sistem akan meminta Anda mengganti password
   - Buat password baru yang kuat

---

### Syarat Password yang Kuat

Untuk keamanan akun Anda, password harus memenuhi kriteria:

✅ **Minimal 8 karakter**
✅ **Kombinasi huruf besar dan kecil** (A-Z, a-z)
✅ **Minimal 1 angka** (0-9)
✅ **Minimal 1 karakter khusus** (!@#$%^&*)

❌ **Hindari**:
- Tanggal lahir atau nama yang mudah ditebak
- Password yang sama dengan username
- Password yang pernah digunakan sebelumnya
- Kata-kata umum dari kamus

**Contoh Password Kuat**: `eSchool2026!@`

### Tips Keamanan Password

#### 1. Jangan Bagikan Password
- Jangan pernah memberitahu password kepada siapa pun
- Admin TIDAK AKAN pernah meminta password Anda
- Waspada terhadap phishing

#### 2. Gunakan Password Manager
- Simpan password dengan aman menggunakan password manager
- Contoh: LastPass, 1Password, Bitwarden
- Atau gunakan fitur password manager browser

#### 3. Aktifkan Two-Factor Authentication (2FA)
- Jika tersedia, aktifkan 2FA untuk keamanan ekstra
- Hubungi admin untuk mengaktifkan fitur ini

#### 4. Ganti Password Secara Berkala
- Disarankan mengganti password setiap 3-6 bulan
- Terutama jika ada indikasi akun terkompromi

### Troubleshooting

#### Email Reset Tidak Diterima

**Penyebab**:
- Email salah atau tidak terdaftar
- Email masuk ke folder Spam
- Server email sedang bermasalah

**Solusi**:
1. Periksa folder Spam/Junk
2. Tambahkan noreply@eschool.ac.id ke whitelist
3. Tunggu 5-10 menit (email mungkin delay)
4. Coba ulangi proses reset
5. Hubungi admin jika tetap tidak menerima

#### Link Reset Expired

**Penyebab**: Link reset berlaku 1 jam

**Solusi**:
1. Ulangi proses reset password dari awal
2. Segera klik link setelah menerima email
3. Jangan tutup tab browser saat proses reset

#### Password Baru Ditolak

**Penyebab**: Password tidak memenuhi kriteria

**Solusi**:
1. Pastikan password minimal 8 karakter
2. Gunakan kombinasi huruf, angka, dan simbol
3. Jangan gunakan password lama
4. Lihat pesan error untuk detail spesifik

### Keamanan Akun

#### Tanda-Tanda Akun Terkompromi

Segera ganti password jika:
- Ada aktivitas yang tidak Anda lakukan
- Menerima notifikasi login dari lokasi asing
- Password tiba-tiba berubah tanpa Anda ketahui
- Data pribadi berubah tanpa izin

#### Langkah Jika Akun Diretas

1. **Segera Reset Password**
   - Gunakan metode reset di atas
   - Buat password yang benar-benar baru

2. **Hubungi Admin**
   - Laporkan kejadian ke admin sekolah
   - Minta review aktivitas akun

3. **Periksa Data**
   - Cek apakah ada data yang berubah
   - Verifikasi informasi pribadi Anda

4. **Ganti Password Lain**
   - Jika menggunakan password yang sama di tempat lain
   - Segera ganti semua password tersebut
