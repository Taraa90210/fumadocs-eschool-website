Jika Anda mengalami kendala saat login ke eSchool, silakan ikuti panduan troubleshooting berikut.

---

### Solusi Umum

#### 1. Periksa Kredensial
**Masalah**: Username atau password salah

**Solusi**:
- Pastikan username ditulis dengan benar (case-sensitive)
- Periksa apakah Caps Lock aktif
- Pastikan tidak ada spasi di awal atau akhir
- Coba copy-paste password jika tersimpan

#### 2. Verifikasi Koneksi Internet
**Masalah**: Tidak ada koneksi internet

**Solusi**:
- Periksa koneksi WiFi atau data seluler
- Coba buka website lain untuk memastikan internet aktif
- Restart router jika menggunakan WiFi
- Pindah ke jaringan yang lebih stabil

#### 3. Clear Browser Cache
**Masalah**: Halaman login tidak loading dengan benar

**Solusi**:
1. Buka pengaturan browser
2. Cari opsi "Clear browsing data" atau "Hapus data penjelajahan"
3. Pilih "Cookies" dan "Cached images and files"
4. Klik "Clear data" atau "Hapus data"
5. Restart browser dan coba login lagi

#### 4. Gunakan Browser yang Didukung
**Masalah**: Browser tidak kompatibel

**Browser yang Direkomendasikan**:
- Google Chrome (versi terbaru)
- Mozilla Firefox (versi terbaru)
- Microsoft Edge (versi terbaru)
- Safari (untuk Mac/iOS)

**Solusi**:
- Update browser ke versi terbaru
- Atau gunakan browser alternatif yang didukung

### Masalah Spesifik

#### Pesan Error: "Invalid Credentials"
**Penyebab**: Username atau password salah

**Solusi**:
1. Periksa kembali username dan password
2. Gunakan fitur "Lupa Password" jika perlu
3. Hubungi admin sekolah untuk verifikasi akun

#### Pesan Error: "Account Locked"
**Penyebab**: Terlalu banyak percobaan login gagal

**Solusi**:
1. Tunggu 15-30 menit sebelum mencoba lagi
2. Atau hubungi admin sekolah untuk unlock akun
3. Reset password jika perlu

#### Pesan Error: "Session Expired"
**Penyebab**: Sesi login sudah kadaluarsa

**Solusi**:
1. Refresh halaman (F5 atau Ctrl+R)
2. Login ulang dengan kredensial Anda
3. Pastikan tidak membuka multiple tab dengan akun yang sama

#### Halaman Login Tidak Muncul
**Penyebab**: Masalah jaringan atau server

**Solusi**:
1. Periksa koneksi internet
2. Coba akses dari jaringan lain
3. Tunggu beberapa saat jika server sedang maintenance
4. Hubungi IT support jika masalah berlanjut

---

### Jaringan Sekolah

#### Akses dari Dalam Sekolah
**Catatan**: Beberapa sekolah membatasi akses ke port tertentu

**Solusi**:
- Pastikan Anda terhubung ke WiFi sekolah yang authorized
- Hubungi IT support jika tidak bisa akses dari jaringan sekolah
- Gunakan jaringan alternatif jika diizinkan

#### Akses dari Luar Sekolah
**Catatan**: Sistem dapat diakses dari mana saja dengan internet

**Solusi**:
- Gunakan koneksi internet yang stabil
- Pastikan tidak ada firewall yang memblokir akses
- Gunakan VPN jika diminta oleh sekolah

---

### Troubleshooting Lanjutan

#### Disable Browser Extensions
Beberapa extension dapat mengganggu login:
1. Buka mode Incognito/Private browsing
2. Coba login di mode tersebut
3. Jika berhasil, disable extension satu per satu untuk menemukan yang bermasalah

#### Check System Time
Waktu sistem yang salah dapat menyebabkan masalah:
1. Pastikan tanggal dan waktu perangkat Anda benar
2. Aktifkan "Set time automatically"
3. Sinkronkan dengan server waktu internet

#### Try Different Device
Jika masalah persisten:
1. Coba login dari perangkat lain (HP, tablet, komputer lain)
2. Jika berhasil, masalah ada di perangkat pertama
3. Jika tetap gagal, masalah mungkin di akun Anda
